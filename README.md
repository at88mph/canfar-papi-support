# Presentation for Permissions API (PAPI) support in CANFAR (Skaha & Cavern)

## Running it
Use Docker to start a web server:
```sh
git clone https://github.com/at88mph/canfar-papi-support.git
cd canfar-papi-support/docs
python3 -m http.server 8080
```

Then visit http://localhost:8080.  The end slash is important.
